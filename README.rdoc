= kerio-caldav

http://github.com/athlite/kerio-caldav

== DESCRIPTION

A Ruby interface to Kerio Mail server calendars
	
== FEATURES

* List a user calendars.

== SYNOPSIS
Setup

	KerioCaldav::Config.config = {
		'url' => 'https://mail.example.com/path/to-calendars',
		'username' =>  'username',
		'password' => 'password'
	}
	
To get calendars of a specific user
	
	calendars = KerioCaldav::Kerio.calendars('username')
	
	calendars.each do |cal|
		cal.events.each do |event|
			p event
		end
	end

== REQUIREMENTS

* icalendar
* curl must be installed

== INSTALL

* sudo gem install kerio-caldav
